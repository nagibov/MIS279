üîë What is a Token?

A token is like a digital ticket proving the user is logged in

Example JWT Token:

eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJ1c2VySWQiOjEyMywiaWF0IjoxNjE2MjM5MDIyfQ.SflKxwRJSMeKKF2QT4fwpMeJf36POk6yJV_adQssw5c

What Does the Token Contain?

JWT has 3 parts (separated by dots):

  • Header: Algorithm type (HS256, RS256)
  • Payload: User data (userId, email, role, expiration)
  • Signature: Encrypted hash to verify authenticity
// Decoded payload example:
{
  "userId": 123,
  "email": "user@example.com",
  "role": "customer",
  "iat": 1616239022,  // issued at
  "exp": 1616325422   // expires at
}

Why is it Secure?

  • Signature verification: Server validates token wasn't tampered with
  • Secret key: Only server knows the secret to create valid signatures
  • Expiration: Tokens expire automatically (e.g., 24 hours)
  • Stateless: No need to store sessions on server
  • Cannot be modified: Any change breaks the signature

Store Token on Frontend:

// Save token after successful login
localStorage.setItem("token", data.token);

// Retrieve token for API requests
const token = localStorage.getItem("token");

// Include in API request headers
fetch("/api/profile", {
  headers: { "Authorization": `Bearer ${token}` }
});

üõí Cart Implementation

// Load cart from localStorage on page load
const cart = JSON.parse(localStorage.getItem("cart")) || [];

// Add item to cart
function addToCart(product) {
  const existing = cart.find(item => item.id === product.id);
  
  if (existing) {
    existing.qty += 1; // Increment quantity
  } else {
    cart.push({...product, qty: 1}); // Add new item
  }
  
  saveCart();
}

// Remove item from cart
function removeFromCart(productId) {
  const index = cart.findIndex(item => item.id === productId);
  cart.splice(index, 1);
  saveCart();
}

// Save cart to localStorage
function saveCart() {
  localStorage.setItem("cart", JSON.stringify(cart));
  updateCartUI();
}

üîé Search & Filter (Using SQL)

Search Books by Title (Bookstore API):

// Frontend sends search query
const searchText = "Harry Potter";
fetch(`/api/books/search?q=${searchText}`);

// Backend SQL query (controllers/books.js)
export async function searchBooks(req, res) {
  const { q } = req.query;
  const result = await pool.request()
    .input("search", sql.NVarChar, `%${q}%`)
    .query(`SELECT b.BookID, b.Title, b.Price, b.Genre, 
             a.FirstName + ' ' + a.LastName AS Author
      FROM Books b
      JOIN Author_Book ab ON b.BookID = ab.BookId
      JOIN Authors a ON ab.AuthorId = a.AuthorID
      WHERE b.Title LIKE @search OR a.LastName LIKE @search`);
  res.json(result.recordset);
}

Filter by Price Range:

// Frontend sends min/max price
fetch(`/api/books?minPrice=10&maxPrice=25`);

// Backend SQL query
const { minPrice, maxPrice } = req.query;
const result = await pool.request()
  .input("min", sql.Decimal(10,2), minPrice)
  .input("max", sql.Decimal(10,2), maxPrice)
  .query(`SELECT * FROM Books WHERE Price BETWEEN @min AND @max`);

Filter by Genre:

// Frontend genre filter
fetch(`/api/books?genre=Fantasy`);

// Backend SQL query
const { genre } = req.query;
const result = await pool.request()
  .input("genre", sql.NVarChar, genre)
  .query(`SELECT * FROM Books WHERE Genre = @genre ORDER BY Title`);

Sort Books (SQL ORDER BY):

// Price: Low to High
SELECT * FROM Books ORDER BY Price ASC

// Price: High to Low
SELECT * FROM Books ORDER BY Price DESC

// Alphabetical by Title
SELECT * FROM Books ORDER BY Title ASC

‚ù§Ô∏è Wishlist Implementation

Simple Wishlist Logic:

let wishlist = JSON.parse(localStorage.getItem("wishlist")) || [];

function addToWishlist(productId) {
  if (!wishlist.includes(productId)) {
    wishlist.push(productId);
    localStorage.setItem("wishlist", JSON.stringify(wishlist));
    updateWishlistUI();
  }
}

function removeFromWishlist(productId) {
  wishlist = wishlist.filter(id => id !== productId);
  localStorage.setItem("wishlist", JSON.stringify(wishlist));
  updateWishlistUI();
}

function isInWishlist(productId) {
  return wishlist.includes(productId);
}
